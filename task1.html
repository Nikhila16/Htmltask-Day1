Tag Definition Example
<html> </html> Defines a Html Document This File
<head> </head> Used to give titles and external links like .css, .js Gives title and Validates the styles, properties etc.,
<title> </title> Gives the title Gives the webpage a specific name
<body> </body> It is where the webpage content is given Displays the content of the webpage
headings(h1,h2..h6)- <h1> </h1> Used for headings Gives the default heading style to content based on the heading tag
<img src="url"> Inserts img Shows img on the webpage
<a href src="url"> </a> Links to the source Shows the page of entered link
<p> </p> Used to enter the content as paragraph Shows the paragraph on the webpage
